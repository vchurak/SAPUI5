Testing Overview

This is an overview page of various ways to test the Shop app during development.
Choose one of the options below to launch the app in the WebIDE preview.

  • test/testFLP.html - start the app in the SAP Fiori launchpad sandbox (with the real backend server). Note: you'll need a properly configured RefAppsBackend destination in this cloud account.
  • test/testFLPService.html - start the app in the SAP Fiori launchpad sandbox (with local mock data)
